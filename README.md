## 🎬 bilibili-subtitle

**从 Bilibili 视频中自动提取字幕内容，**  支持**有字幕 / 无字幕**视频，可输出纯文本或带时间戳文本。

> 针对 B 站大量无字幕视频的场景，工具会按需下载并使用  **FFmpeg 等音视频组件**进行语音识别与字幕生成。
